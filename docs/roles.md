# Team Roles

## 윤예나: 데이터셋 / 학습 담당

- PPE 데이터셋 수집
- 클래스명 정리
- 라벨링 품질 확인
- data.yaml 확인
- YOLO 학습 실행
- best.pt 선정
- 성능 결과 정리

## 이수아: 실시간 감지 담당

- YOLO 모델 로드
- 웹캠 또는 영상 파일 연결
- 프레임 단위 추론
- bounding box 표시
- class / confidence 추출
- 감지 결과를 JSON 형태로 정리

## 신채원: 위험 판단 담당

- 안전모 미착용 판단
- 안전조끼 미착용 판단
- 위험구역 좌표 설정
- 작업자 중심점이 위험구역 안에 있는지 판정
- 위험도 점수 계산
- 위반 시 캡처 트리거 조건 작성
