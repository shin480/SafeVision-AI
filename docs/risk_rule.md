# Risk Level Rule

## 1. 위험도 산정 기준

위험도 점수는 감지된 조건에 따라 누적하여 계산한다.

| 조건                          |  점수 |
| --------------------------- | --: |
| 위험구역 진입                     | +30 |
| 안전모 미착용 (`no_helmet`)       | +40 |
| 안전조끼 미착용 (`no_safety_vest`) | +20 |

---

## 2. 위험도 등급 기준

누적된 위험도 점수에 따라 위험 등급을 분류한다.

|   점수 범위 | 등급       | 상태    |
| ------: | -------- | ----- |
|  0 ~ 29 | SAFE     | 정상    |
| 30 ~ 59 | WARNING  | 주의    |
| 60 ~ 79 | DANGER   | 위험    |
|   80 이상 | CRITICAL | 매우 위험 |

---

## 3. 이벤트 발생 기준

위험도 등급에 따라 로그 기록, 캡처 저장, 관리자 알림 여부를 결정한다.

| 등급       | 처리 내용                                |
| -------- | ------------------------------------ |
| SAFE     | 별도 처리 없음                             |
| WARNING  | 로그 기록                                |
| DANGER   | 로그 기록, 위반 상태 표시, 캡처 저장               |
| CRITICAL | 로그 기록, 위반 상태 표시, 캡처 저장, 관리자 화면 위험 알림 |

---

## 4. 설계 의도

위험구역은 작업 수행을 위해 진입이 필요한 공간이므로, 위험구역 진입 자체를 즉시 위험 상황으로 판단하지 않는다.

작업자가 위험구역에 진입하더라도 안전모와 안전조끼를 정상적으로 착용한 경우에는 정상 작업으로 간주한다.

반면, 위험구역 내에서 안전모 미착용 또는 안전조끼 미착용 상태가 확인될 경우 위험도 점수를 증가시켜 작업자의 안전사고 가능성을 평가하도록 설계하였다.
