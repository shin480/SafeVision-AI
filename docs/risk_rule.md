4. 설계 의도
위험구역은 작업 수행을 위해 진입이 필요한 공간이므로, 위험구역 진입 자체를 위험으로 판단하지 않는다.
작업자가 위험구역에 진입하더라도 PPE(안전모, 안전조끼)를 정상 착용한 경우에는 정상 작업으로 간주한다.
반면 위험구역 내에서 PPE 미착용 상태가 확인될 경우 위험도를 증가시켜 작업자 안전사고 가능성을 평가하도록 설계하였다.
